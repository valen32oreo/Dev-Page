Laravel 11 Project

Valen Gadget Rent

Valen Gadget Rent is a web-based gadget rental system developed using Laravel 11. The platform allows users to browse available gadgets, place rental orders, and manage transactions efficiently through a modern and user-friendly interface.

Key Features

  • User authentication & authorization
  • Gadget catalog & filtering
  • Online rental transactions
  • Admin dashboard management

Technologies

  • Laravel 11
  • MySQL Database
  • Blade Template Engine
  • Tailwind CSS
Coffee Shop Website

Beanology

Beanology is a coffee shop website designed to showcase menu offerings, brand identity, and store information. The website focuses on clean visuals, smooth layout flow, and a warm aesthetic that reflects a modern café atmosphere.

Main Features

  • Interactive menu display
  • Brand-focused landing page
  • Responsive layout
  • Contact & location section

Design Approach

  • Minimalist UI
  • Warm color palette
  • Modern typography
  • Mobile-first design
